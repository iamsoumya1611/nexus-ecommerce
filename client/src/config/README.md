# Configuration

This directory contains centralized configuration files for the application.

## Color Palette

The color palette is centralized in `colors.js` and can be easily modified to change the entire application theme.

### Files

- `colors.js` - Centralized color definitions
- `COLOR_PALETTE.md` - Documentation for the color system

### How to Change Colors

1. Edit the values in `colors.js`
2. The changes will automatically apply to:
   - Tailwind CSS classes
   - CSS variables
   - All components using these colors

This system ensures consistency and makes theme changes simple and maintainable.
